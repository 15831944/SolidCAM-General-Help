CAM Settings

The SolidCAM Settings dialog box enables you to change the path to related applications, units, tolerances, default machines, editors, and color settings for SolidCAM. You can easily view and edit the current program preferences and options.

Managing CAM Settings

SolidCAM enables you to save the CAM settings as an external file and then load the saved file into SolidCAM. Furthermore, SolidCAM enables you to reset the customized settings to their default values.
