System Requirements

Supported Operating Systems

Microsoft® Windows 7 x32/x64 Professional and Ultimate editions

Microsoft® Windows Vista x32/x64 Business and Ultimate editions with Service Pack1

Microsoft® Windows XP Professional with Service Pack 2 or 3

CPU

Intel® Core™, Intel® Core™2 Duo, Intel® Core™ Quad, Quad-core Intel® Xeon®, AMD Phenom™, AMD Phenom™ II, AMD Athlon™ X2 Dual-Core - class processor

RAM

2 GB RAM or more (4 GB or more for x64 operating system is recommended for large
CAM-Parts machining)

Video

An OpenGL workstation graphics card (512 MB RAM recommended) and latest driver

Other

  1. Mouse or other pointing device

  2. CD drive

  3. Internet Explorer version 6 if you are using the SolidCAM Online Help

  4. For viewing the Getting Started manuals, Adobe Acrobat version 9 or higher is recommended.
