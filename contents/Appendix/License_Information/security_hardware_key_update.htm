SolidCAM Security Hardware Key Update

In some cases, SolidCAM requires the update of SolidCAM security hardware keys. To update the security hardware key, do the following:

 

 

  1. Make sure that the SolidCAM security hardware key is connected to the computer and that SolidCAM is closed.

  2. Extract the ZIP-archive <security hardware key number>.zip supplied by SolidCAM into the ..\SolidCAM2019\Util folder.

  3. Load the remote update program by clicking Start > SolidCAM2019 > Update dongle (or run RemoteKeyUpdate.exe from SolidCAM Util folder). 

  4. Browse for the license file (*.lis).

          

  1. Click Perform Update.
