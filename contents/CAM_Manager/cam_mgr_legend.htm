CAM tree entities use the following conventions and symbols:

Convention/Symbol

Description

The synchronization mark displayed next to the geometry icon means that a misfit is determined between the SOLIDWORKS model and the SolidCAM geometry. It can also appear if there is a gap or overlapping entities. To resolve the issue, perform the Synchronization check. 

The exclamation mark means that the geometry cannot be updated. You have to edit this geometry and manually update problematic chains.

The grayed out name of the operation means that the operation is suppressed.

The black rectangle around the operation name means that the GCode of the operation is suppressed.

The lock symbol displayed on the operation icon means that the operation is locked during the parallel calculation.

The red square displayed on the operation icon means that the operation contains an error, for example, the submachine has an empty table (no stock has been loaded). Hover the mouse pointer over the operation to get an exact explanation.

The digit 4 in a square displayed on the operation icon means that the operation is transformed using the 4-Axis option.

The letter R in a square displayed on the operation icon means that the operation is transformed using the Rotation option.

The letter M in a square displayed on the operation icon means that the operation is transformed using the Mirror option.

The letter T in a square displayed on the operation icon means that the operation is transformed using the Translate option.

The Coordinate system symbol in a square displayed on the operation icon means that the operation is transformed using the Coordsys option.

The letter X in a square displayed on the operation icon means that the operation is transformed using more than one transformation option.

The cyan circle displayed on the operation icon means that the Milling operation calculates and updates the material boundary to be used in subsequent turning operations.

The green diamond symbol instead of the tool path check box before the operation icon means that the operation is modified with the Tool Path Editor.

The asterisk symbol instead of the tool path check box before the operation icon means that the operation is not calculated.

The red color of the font means that the operation is incomplete.

You can use the tool path display check box to toggle between showing and hiding
the tool path.
